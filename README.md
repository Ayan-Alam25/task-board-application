## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm (v8 or higher)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Ayan-Alam25/task-board-application.git
   cd task-board-application
   ```
2. Install dependencies:
    ```bash
    npm install
    ```

3.  Run the development server:

    ```bash
    npm run dev
    ```
